# LSS 个人博客

## 📖 项目介绍
这是一个基于Hexo + Butterfly主题的个人博客，包含丰富的娱乐资源和内容。

## 🚀 功能特色
- 🎵 音乐收藏 - 免费音乐网站和华语热歌推荐
- 🎬 电影收藏 - 免费电影平台和热门推荐
- 🌸 动漫番剧 - 经典番剧推荐和免费观看网站
- 📚 漫画世界 - 漫画平台和国漫日漫分类
- 📖 小说阅读 - 免费小说资源和网文经典
- 🎧 ASMR放松 - 放松音频资源和使用指南

## 🛠️ 技术栈
- **框架**: Hexo 7.3.0
- **主题**: Butterfly 5.3.4
- **部署**: GitHub Pages
- **构建**: GitHub Actions
