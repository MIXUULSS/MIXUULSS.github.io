# LSS的个人博客

🌟 **一个专注于技术分享和生活记录的个人博客**

## 📋 项目介绍

这个博客基于 [Hexo](https://hexo.io/) 静态网站生成器构建，使用了精美的 [Butterfly](https://github.com/jerryc127/hexo-theme-butterfly) 主题。

### 🎨 博客特色

- **响应式设计** - 完美适配各种设备
- **优美主题** - Butterfly主题，界面美观
- **内容丰富** - 涵盖多个兴趣领域
- **免费托管** - 部署在GitHub Pages

### 📱 页面内容

| 页面 | 内容描述 |
|------|----------|
| 🎵 音乐天地 | 免费音乐网站推荐和歌曲精选 |
| 🎬 电影世界 | 最新电影推荐和观看资源 |
| 🎌 动漫番剧 | 热门番剧和免费观看网站 |
| 📚 漫画推荐 | 国内外优秀漫画和阅读资源 |
| 📖 小说天地 | 免费小说网站和热门网文 |
| 🎧 ASMR助眠 | 助眠资源和放松指导 |
| 👋 关于我 | 个人介绍和博客理念 |
| 🔗 友情链接 | 优质网站资源分享 |

## 🛠️ 技术栈

- **Hexo 5.5.4** - 静态网站生成器
- **Butterfly 5.3.4** - 精美主题
- **Node.js** - JavaScript运行环境
- **GitHub Pages** - 免费静态网站托管
- **Git** - 版本控制系统

## 🚀 快速开始

### 本地运行

1. **克隆仓库**
   ```bash
   git clone https://github.com/MIXUULSS/MIXUULSS.github.io.git
   cd MIXUULSS.github.io
   ```

2. **安装依赖**
   ```bash
   npm install
   ```

3. **启动本地服务器**
   ```bash
   hexo clean
   hexo generate
   hexo server
   ```

4. **访问博客**
   ```
   http://localhost:4000
   ```

### 发布新文章

```bash
# 创建新文章
hexo new "文章标题"

# 编辑文章
vim source/_posts/文章标题.md

# 生成并部署
hexo clean
hexo generate
hexo deploy
```
