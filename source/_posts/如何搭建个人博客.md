## 📝 搭建个人博客完整教程

想拥有一个属于自己的博客吗？这篇教程将教你如何使用Hexo快速搭建个人博客。

### 🛠️ 准备工作

#### 必需工具
- [Node.js](https://nodejs.org/) - JavaScript运行环境
- [Git](https://git-scm.com/) - 版本控制工具
- [GitHub账号](https://github.com/) - 代码托管平台

#### 知识要求
- 基础的命令行操作
- 了解Git基本概念
- 有HTML/CSS基础更佳

### 🚀 开始搭建

#### 1. 安装Hexo
```bash
npm install -g hexo-cli
```

#### 2. 初始化项目
```bash
hexo init my-blog
cd my-blog
npm install
```

#### 3. 本地预览
```bash
hexo server
```
访问 `http://localhost:4000` 查看效果

### 🎨 选择主题

Hexo有很多优秀的主题，推荐：

- **Butterfly** - 美观的功能丰富的主题
- **NexT** - 简洁实用的主题
- **Volantis** - 现代化设计主题

安装Butterfly主题：
```bash
npm install hexo-theme-butterfly
```

### 📖 发布文章

#### 创建新文章
```bash
hexo new "文章标题"
```

#### 文章格式
```markdown
---
title: 文章标题
date: 2025-03-02 14:00:00
tags:
  - 标签1
  - 标签2
categories:
  - 分类
---

文章内容...
```

### 🌐 部署到GitHub

#### 1. 创建仓库
在GitHub创建 `username.github.io` 仓库

#### 2. 安装部署工具
```bash
npm install hexo-deployer-git --save
```

#### 3. 配置部署
在 `_config.yml` 中添加：
```yaml
deploy:
  type: git
  repo: https://github.com/username/username.github.io.git
  branch: main
```

#### 4. 部署
```bash
hexo clean
hexo generate  
hexo deploy
```

### 📸 常用命令

```bash
# 清理缓存
hexo clean

# 生成静态文件
hexo generate

# 本地预览
hexo server

# 部署到服务器
hexo deploy

# 一键部署
hexo clean && hexo generate && hexo deploy
```

### 💡 优化建议

#### 性能优化
- 开启Gzip压缩
- 使用CDN加速
- 图片懒加载
- 开启缓存

#### SEO优化
- 完善文章标题
- 添加meta描述
- 设置合适的URL结构
- 提交搜索引擎

### 🔧 常见问题

#### Q: 如何自定义主题？
A: 修改主题配置文件，或创建自定义CSS。

#### Q: 如何添加评论功能？
A: 集成Valine、Gitalk等评论系统。

#### Q: 如何实现统计功能？
A: 集成Google Analytics、百度统计等。
