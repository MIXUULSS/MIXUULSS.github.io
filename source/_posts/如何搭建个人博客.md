# 🚀 如何搭建个人博客

想拥有自己的个人博客吗？今天我来分享一个完整的博客搭建方案！

## 📋 前期准备

### 🛠️ 所需工具
- **Node.js** - JavaScript运行环境
- **Git** - 版本控制系统
- **GitHub账号** - 代码托管和免费域名
- **文本编辑器** - VS Code推荐
- **基础HTML/CSS知识** - 有帮助但不是必需的

### 📚 学习资源
- [Hexo官方文档](https://hexo.io/)
- [Butterfly主题文档](https://butterfly.js.org/)
- [GitHub Pages教程](https://pages.github.com/)

## 🎯 搭建步骤

### 1️⃣ **安装Hexo**
```bash
# 全局安装Hexo CLI
npm install -g hexo-cli

# 创建博客目录
hexo init my-blog
cd my-blog

# 安装依赖
npm install
```

### 2️⃣ **安装Butterfly主题**
```bash
# 克隆主题
git clone https://github.com/jerryc127/hexo-theme-butterfly.git themes/butterfly

# 安装渲染器
npm install hexo-renderer-pug hexo-renderer-stylus --save
```

### 3️⃣ **配置主题**
在 `_config.yml` 中设置：
```yaml
theme: butterfly
```

### 4️⃣ **本地预览**
```bash
# 清理缓存
hexo clean

# 生成静态文件
hexo generate

# 启动本地服务器
hexo server
```

## 🎨 主题配置

### 📝 **基本设置**
编辑 `_config.butterfly.yml`：

```yaml
# 导航菜单
menu:
  首页: / || fa fa-home
  关于: /about/ || fa fa-user
  归档: /archives/ || fa fa-archive
  标签: /tags/ || fa fa-tags

# 个人信息
avatar:
  img: /img/avatar.png
  effect: true

# 社交媒体
social:
  fa fa-github: https://github.com/username || Github
  fa fa-envelope: mailto:email@example.com || Email
```

### 🎯 **页面创建**
```bash
# 创建关于页面
hexo new page about

# 创建分类页面
hexo new page categories

# 创建标签页面
hexo new page tags
```

## 📝 内容创作

### 📄 **创建文章**
```bash
# 创建新文章
hexo new "文章标题"

# 创建草稿
hexo new draft "草稿标题"

# 发布草稿
hexo publish "草稿标题"
```

### 🏷️ **文章格式**
```markdown
---
title: 文章标题
date: 2025-03-02 12:00:00
tags:
  - 标签1
  - 标签2
categories:
  - 分类名称
---

# 文章内容
...
```

## 🌐 部署到GitHub Pages

### 1️⃣ **创建仓库**
- 在GitHub创建 `username.github.io` 仓库
- 设置为公开仓库

### 2️⃣ **安装部署插件**
```bash
npm install hexo-deployer-git --save
```

### 3️⃣ **配置部署**
在 `_config.yml` 中添加：
```yaml
deploy:
  type: git
  repo: https://github.com/username/username.github.io.git
  branch: master
```

### 4️⃣ **一键部署**
```bash
hexo deploy
```

## 🔍 常见问题

### ❓ **部署失败**
- 检查SSH密钥配置
- 确认仓库权限
- 验证分支设置

### ❓ **主题不显示**
- 确认主题名称正确
- 检查文件路径
- 重新生成静态文件

### ❓ **样式加载失败**
- 清理浏览器缓存
- 检查CDN链接
- 验证主题文件完整性
