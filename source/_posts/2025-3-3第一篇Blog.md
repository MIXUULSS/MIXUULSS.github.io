## 基本原理
### 1. 动画三要素
```css
/* 过渡动画示例 */
.button {
  transition: 
    transform 0.3s ease-in-out,
    opacity 0.2s linear;
}
```

### 2. 关键帧动画
```css
@keyframes slide-in {
  0% {
    transform: translateX(-100%);
  }
  90% {
    transform: translateX(5%);
  }
  100% {
    transform: translateX(0);
  }
}
```

## 实战案例
### 加载动画实现
```html
<div class="loader">
  <div class="dot delay-1"></div>
  <div class="dot delay-2"></div>
  <div class="dot delay-3"></div>
</div>

<style>
.loader .dot {
  animation: bounce 1.4s infinite;
}
@keyframes bounce {
  0%, 80%, 100% { 
    transform: scale(0);
  }
  40% {
    transform: scale(1);
  }
}
</style>
```

## 性能优化
| 优化策略         | 实施方法                     | 效果提升 |
|------------------|------------------------------|----------|
| 减少重排         | 使用`transform`代替`left/top` | 30%↑     |
| 硬件加速         | 添加`will-change`属性         | 45%↑     |
| 复合层管理       | 控制`z-index`层级             | 25%↑     |
